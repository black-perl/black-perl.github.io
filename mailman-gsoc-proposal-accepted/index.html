It's gsoc'15 with GNU Mailman :-)

29 Apr 2015

Ah ! there came the moment I have been waiting for the past two years. The list of accepted projects got published on the Google Summer of Code website. And, after two months of consistent hard work, I made it. I will be working with an open source organisation called as GNU Mailman, which is a mailing list management software.

Coming back to my project. The Mailman project is primarily in Python. I will be going to port mailman.client, an existing Python REST client for communicating with mailman-core to a Node.js package i.e a npm package. More on this here.

I will be blogging throughout the summers, as I keep on hacking with Mailman ;-) Hang around !!
